## 安装说明

### 系统要求

- Android 5.0 (API Level 21) 或更高版本
- 支持 Android TV 系统
- 1920x1080 或更高分辨率
- 网络连接 (WiFi 或有线网络)

### 安装步骤

1. 从官方渠道获取 APK 安装包
2. 在 Android TV 上打开「设置」->「设备偏好设置」->「安全和限制」
3. 启用「未知来源」选项
4. 使用 USB 存储设备或应用商店安装 APK
5. 安装完成后,在应用列表中找到 LunaTV 图标启动

## 遥控器操作

### 基本按键说明

| 按键 | 功能 |
|------|------|
| 方向键 (↑↓←→) | 移动焦点选择内容 |
| 确认键 (OK/Enter) | 打开或选择项目 |
| 返回键 | 返回上一级或退出 |
| Home 键 | 返回主屏幕 |
| 菜单键 | 显示选项菜单 |

### 导航技巧

1. **快速滚动**: 长按方向键可以快速滚动
2. **焦点指示器**: 当前选中的项目会有高亮边框
3. **页面切换**: 使用返回键可以在不同页面间切换
4. **播放控制**: 在播放页面使用方向键控制播放

### 操作流程示例

1. 启动应用后,使用方向键选择影视分类
2. 按 OK 键进入分类,浏览影视列表
3. 使用方向键选择影片,按 OK 键播放
4. 播放时按方向键可以快进/快退
5. 按 OK 键暂停/继续播放
6. 按返回键返回列表或主界面

## 基本功能

### 首页

- **Banner 轮播**: 展示热门推荐影片
- **分类导航**: 电影、电视剧、综艺、动漫、短剧
- **快速访问**: 按 OK 键直接播放推荐内容

### 搜索

1. 进入搜索页面
2. 使用方向键移动到输入框
3. 按屏幕虚拟键盘输入搜索关键词
4. 按 OK 键执行搜索
5. 浏览搜索结果并选择播放

### 播放

- **多播放源**: 支持切换不同的播放源
- **集数选择**: 电视剧和动漫支持选择集数
- **播放记录**: 自动保存观看进度,下次继续播放
- **全屏播放**: 默认全屏显示,享受影院体验

### 直播

- **频道列表**: 浏览所有可用的直播频道
- **频道切换**: 按 OK 键快速切换频道
- **节目单**: 查看当前和即将播出的节目

### 短剧

- **短剧专区**: 浏览热门短剧内容
- **快速播放**: 按 OK 键直接开始播放

### 用户中心

- **登录注册**: 使用账号登录享受更多功能
- **收藏管理**: 收藏喜欢的影片
- **历史记录**: 查看观看历史
- **积分兑换**: 使用积分兑换会员

### 设置

- **服务器配置**: 自定义后端服务器地址
- **播放设置**: 自动播放下一集等选项
- **清理缓存**: 清除本地缓存数据

## 常见问题

### Q1: 如何登录账号?

A: 进入「我的」页面,选择「登录」选项,按照提示输入账号密码即可。

### Q2: 影片无法播放怎么办?

A: 请检查:
- 网络连接是否正常
- 服务器地址配置是否正确
- 尝试切换播放源
- 确认是否已登录账号

### Q3: 如何退出应用?

A: 长按 Home 键或按返回键多次即可退出应用。

### Q4: 遥控器操作不灵敏?

A: 检查遥控器电池电量,或尝试更换遥控器。

### Q5: 如何切换到其他页面?

A: 使用底部导航栏,按方向键选择对应的标签页,按 OK 键切换。

### Q6: 影片加载慢?

A: 可能原因:
- 网络速度较慢
- 服务器响应慢
- 视频源问题

建议:
- 检查网络连接
- 尝试切换播放源
- 更换网络环境

### Q7: 如何清理缓存?

A: 进入「我的」->「设置」->「缓存管理」,选择清理缓存。

### Q8: 应用闪退怎么办?

A: 请尝试:
- 重启应用
- 清理应用数据
- 重新安装应用
- 检查系统兼容性

### Q9: 支持 4K 分辨率吗?

A: 支持 4K 分辨率,但实际播放质量取决于视频源和网络速度。

### Q10: 如何联系客服?

A: 如遇到技术问题,请通过以下方式联系:
- 提交 GitHub Issue
- 发送邮件至客服邮箱
- 访问官方社区
